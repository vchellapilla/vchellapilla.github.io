projects

In the spring of 2020, I took a class that taught me how to analyze data and then create visualizations that helped relay the data in a comprehensive way. Here are a few published projects I've worked on since then:


"Pennsylvania Counties Impacted by COVID-19"

Updated daily since March 14, 2020

Map depicting counties in Pennsylvania affected by the novel coronavirus. The map shows how many cases have been reported in each county and how many deaths have occurred as well.


"Number of Coronavirus Cases in Pennsylvania by County"

Updated daily since March 22, 2020

Sometimes a map simply isn't enough to be able to compare and contrast the given information. This bar graph shows viewers how deeply each county has been affected in regards to the number of cases and provides a way to compare between all the counties in Pennsylvania


This Website!

April 7, 2020

In the spring of 2020, I took part in a nine-week long coding workshop hosted by the Penn State University Libraries - Code For Her. The workshop was a beginner's course for women and gender-diverse students at Penn State University Park that focused on web development, HTML, CSS and JavaScript. As part of my final project, I created this website from scratch over the course of two days.
